# 📊 E-Commerce Sales & Customer Insights Dashboard (Power BI)

## 📖 Description
This project is an interactive Power BI dashboard built to analyze e-commerce sales performance, customer behavior, and product insights. It helps in understanding revenue trends, customer segmentation, and geographical sales distribution for better decision-making.

---

## 🚀 Features
- Revenue, Quantity, and Avg Unit Price analysis  
- Year-wise and Monthly Sales Trends  
- Customer Insights (Total, Active, Returning Customers)  
- Product-wise performance analysis  
- Location-wise sales distribution  
- Year-over-Year (YoY) growth analysis  
- Top-selling products by quantity  

---

## 🛠️ Tech Stack
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling (Star Schema)  

---

## 📂 Dataset
- customer_dim – customer data  
- product_dim – product data  
- time_dim – date/time data  
- store_dim – store/location data  
- fact_table – sales data
